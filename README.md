
# SAEI Telegram Bot 🚀

Advanced Telegram bot for Egyptian students and productivity.

## Features
- AI Chat Assistant
- Study Planner
- Pomodoro Timer
- Daily Challenges
- Streak System
- Task Manager
- Quiz System
- Admin Broadcast
- Firebase Ready
- Arabic-first UX

## Setup

1. Install requirements:
```bash
pip install -r requirements.txt
```

2. Create `.env`
```env
BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
OPENAI_API_KEY=YOUR_API_KEY
ADMIN_ID=YOUR_TELEGRAM_ID
```

3. Run:
```bash
python bot.py
```
